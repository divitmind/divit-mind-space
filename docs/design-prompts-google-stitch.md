# Google Stitch Image Prompts for Divit MindSpace

Use these prompts in Google Stitch to generate premium, editorial-quality images that match the website's warm, elegant aesthetic.

## Color Reference
- Background: `#FDFBF7` (soft cream)
- Warm accent: `#FFE175` (honey yellow)
- Green accent: `#89A09E` (sage gray-green)
- Deep purple: `#52154E`
- Forest green: `#004540`

## Usage Instructions

1. Copy the prompt text (inside the code blocks)
2. Paste into Google Stitch
3. Generate multiple variations
4. Choose the one that best matches the warm, premium aesthetic
5. Upload to Sanity CMS in the appropriate section

## Design Principles

- **Lighting**: Always natural, soft, golden-hour inspired
- **Color Grading**: Warm cream/honey filter
- **Composition**: Minimalist, intentional negative space
- **Expressions**: Authentic, not forced smiles
- **Environment**: Modern minimalist Indian homes, cream/beige/sage tones
